++ sudo -l (Permite ver los comandos que podemos ejecutar siendo root)
++ URL: https://gtfobins.github.io/#+sudo
--- Nota: el url permite ver un listado de los comandos que pueden abusarse para tener acceso root 
--- Por ejemplo si mediante sudo -l nos aparece que tenemos permisos para ejecutar time, visitando el url vemos que existe un comando vinculado que nos permite tener acceso como root, el cual es: 
---- sudo /usr/bin/time /bin/sh
--- Se da el mismo caso con el comando man, a continuaci√≥n tenemos el equivalente:
---- sudo man man
---- !/bin/sh
